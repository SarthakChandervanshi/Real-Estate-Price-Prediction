# Real-Estate-Price-Prediction
 ##### This is a flutter web app to predict price of home in certain locations.

 ##### Input Parameters: 
 ##### (1) Location
 ##### (2) Total squarefoot area
 ##### (3) BHK config
 ##### (4) No of bathrooms

 ##### Target variable: 
 ##### Price

##### Technologies Used:
- ##### Flutter 3.10.6
- ##### Flask 3.0.0
- ##### Python 3.11.5
- ##### Jupyter Notebook 6.5.4
